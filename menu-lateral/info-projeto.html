🎓 Sobre o Projeto

O que é o Rotas BG?

Rotas BG é um sistema interativo de visualização de rotas de transporte coletivo da cidade de Barra do Garças, desenvolvido como projeto acadêmico para demonstrar o uso de tecnologias web modernas aplicadas a problemas reais de mobilidade urbana.

Objetivo

O projeto visa criar uma ferramenta intuitiva e acessível que permita aos cidadãos:

  • ✅ Visualizar as rotas de ônibus disponíveis
  • ✅ Localizar paradas de ônibus
  • ✅ Acessar informações sobre horários e linhas
  • ✅ Explorar pontos de interesse da cidade
  • ✅ Melhorar a experiência de mobilidade urbana

Tecnologias Utilizadas

Frontend

HTML5, CSS3, JavaScript

Mapa Interativo

Leaflet.js + OpenStreetMap

Roteamento

Leaflet Routing Machine (OSRM)

Design

Responsivo e Moderno

Funcionalidades

  • 🗺️ Mapa Interativo: Visualize a cidade com zoom e pan
  • 🚌 Linhas de Ônibus: Selecione e veja as rotas no mapa
  • 🚏 Paradas: Localize todas as paradas com ícones personalizados
  • 📍 Pontos de Interesse: Conheça locais importantes da cidade
  • 📱 Responsivo: Funciona em desktop, tablet e mobile

Estrutura do Projeto

O projeto segue uma arquitetura modularizada e organizada:

  • assets/: Estilos CSS e scripts JavaScript
  • pontos/: Dados de pontos de parada e pontos de interesse
  • rotas/: Dados das linhas de ônibus
  • menu-lateral/: Páginas modulares do menu

Desenvolvimento

Este projeto demonstra boas práticas de desenvolvimento web:

  • ✨ Modularização de código
  • ✨ Design responsivo
  • ✨ Interface intuitiva
  • ✨ Integração com APIs externas
  • ✨ Accessibilidade
